A web platform built to showcase Georgian art and heritage. Users can browse the catalog, interact with an AI assistant about the collection, and leave comments/likes on their favorite pieces.

#Main Features
AI Art Assistant: A context-aware assistant (Groq Cloud API) that helps users learn about the gallery.
Interactive Catalog: View detailed information about Georgian artworks.
Community Features: Leave comments and "Like" artworks (built with AJAX for a smooth experience).
Admin Dashboard: Full CRUD functionality to add, edit, or delete products.

Built With
Backend:Flask (Python)
Database: SQLite & SQLAlchemy
AI: Groq Llama 3.3
Frontend: Bootstrap 5 & JavaScript (AJAX)
Deployment
The live version is hosted on **Render**. 
(Note: To run locally, you will need a `.env` file with your own `GROQ_API_KEY`.)
